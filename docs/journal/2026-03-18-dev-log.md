# 개발일지 - 수눌음 회원 관리 웹 앱

## 날짜: 2026-03-18

## 프로젝트 개요
- **프로젝트명**: 수눌음 통합 회원 관리 시스템
- **기술 스택**: React + Vite + Tailwind CSS v4 + React Router + Supabase + TanStack Query + Lucide React
- **배포**: Vercel (https://sunuleum-member-app.vercel.app)
- **인증**: Google / 카카오 소셜 로그인 (Supabase Auth)
- **데이터베이스**: Supabase PostgreSQL + RLS

## 기능 요구사항 (v2 - 통합 시스템)
### 회원용
1. Google/카카오 소셜 로그인
2. 프로필 관리 (이름, 연락처)
3. 자녀 관리 (이름, 출생연도, 메모)
4. 프로그램 조회 및 신청 (본인/자녀)
5. 출석 내역 조회 (연도별)
6. 회비 납부 현황 조회 (연도별)

### 관리자용
7. 대시보드 (전체 통계)
8. 회원 관리 (CRUD, 자녀 조회, CSV 내보내기)
9. 출석 관리 (연도별, 유형별)
10. 회비 관리 (월별, 상태별)
11. 프로그램 관리 (CRUD, 신청 현황)
12. 시설 관리 (CRUD)
13. 대관 관리 (승인/거절)

### 공개
14. 외부 기관 대관 신청 (로그인 불필요)

## 아키텍처 결정
- **인증**: Supabase Auth (Google, Kakao OAuth)
- **DB**: Supabase PostgreSQL + Row Level Security
- **상태 관리**: TanStack React Query (서버 상태), Supabase 직접 호출
- **라우팅**: React Router + AuthGuard/AdminGuard
- **레이아웃**: MemberLayout (회원용), AdminLayout (관리자용)

## 프로젝트 구조
```
src/
  lib/
    supabase.js         - Supabase 클라이언트
    auth.jsx            - AuthContext (로그인/프로필 관리)
  components/
    common/             - Modal, StatusBadge, ConfirmDialog, LoadingSpinner
    guards/             - AuthGuard, AdminGuard
    layout/             - MemberLayout, AdminLayout
  pages/
    auth/               - Login, Callback, CompleteProfile
    member/             - Home, Profile, Children, Programs, MyEnrollments, MyAttendance, MyFees
    admin/              - Dashboard, Members, Attendance, Fees, Programs, Facilities, Rentals
    public/             - RentalRequest
  utils/                - csv.js, helpers.js, storage.js
supabase/
  schema.sql            - DB 스키마 + RLS 정책 + 트리거
```
