# 개발일지 - 수눌음 회원 관리 웹 앱

## 날짜: 2026-03-18

## 프로젝트 개요
- **프로젝트명**: 수눌음 회원 관리 시스템
- **기술 스택**: React + Vite + Tailwind CSS v4 + React Router DOM + Lucide React
- **배포 대상**: Vercel
- **데이터 저장**: localStorage (백엔드 없이 프론트엔드 단독)

## 기능 요구사항
1. 회원 등록 (이름, 연락처, 가입일, 활동 상태)
2. 회원 목록 조회 및 검색/필터
3. 회원 정보 수정 및 삭제
4. 출석/활동 기록 관리
5. 회비 납부 현황 관리
6. 대시보드 (전체 회원 수, 활동 회원, 미납 현황 등)
7. CSV 내보내기 기능

## 아키텍처 결정
- **상태 관리**: React Context + useReducer 패턴
- **라우팅**: React Router DOM (BrowserRouter)
- **UI 프레임워크**: Tailwind CSS v4 (Vite 플러그인 방식)
- **아이콘**: Lucide React
- **개발 방식**: 멀티 에이전트 병렬 개발

## 프로젝트 구조
```
src/
  components/Layout.jsx    - 사이드바 레이아웃
  pages/
    Dashboard.jsx          - 대시보드 (통계 카드, 최근 활동)
    Members.jsx            - 회원 CRUD
    Attendance.jsx         - 출석/활동 기록
    Fees.jsx               - 회비 납부 관리
  context/AppContext.jsx   - 전역 상태 (members, attendance, fees)
  utils/
    storage.js             - localStorage 래퍼
    csv.js                 - CSV 내보내기
    helpers.js             - 유틸 함수 (ID 생성, 날짜/통화 포맷)
```
