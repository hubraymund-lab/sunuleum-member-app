# 개발일지 - 장난감 대여 기능 추가

## 날짜: 2026-03-22

## 프로젝트 개요
- **프로젝트명**: 수눌음 통합 회원 관리 시스템
- **기술 스택**: React + Vite + Tailwind CSS v4 + React Router + Supabase + TanStack Query + Lucide React
- **배포**: Vercel (https://sunuleum-member-app.vercel.app)

## 작업 내용

### 1. 모바일 반응형 UI 적용 (이전 작업)
- MemberLayout: 모바일 햄버거 메뉴 + 슬라이드 드로어 네비게이션 추가
- MyAttendance/MyFees: 테이블을 모바일 카드 리스트로 전환
- MyEnrollments: flex 레이아웃 모바일 대응
- 전체 회원 페이지 반응형 패딩/타이포그래피 적용

### 2. 장난감 대여 기능 추가
- DB 스키마: toys, toy_rentals 테이블 + RLS 정책
- 회원용 페이지:
  - Toys.jsx: 장난감 목록 조회 및 대여 신청
  - MyToyRentals.jsx: 내 대여 내역 조회 및 반납
- 관리자용 페이지:
  - AdminToys.jsx: 장난감 CRUD 관리
  - AdminToyRentals.jsx: 대여 현황 관리 (반납 처리, 연체 처리)
- 라우팅 업데이트: App.jsx에 새 라우트 추가
- 네비게이션 업데이트: MemberLayout, AdminLayout에 메뉴 추가
- StatusBadge에 장난감 관련 상태 스타일 추가

### 3. 새 DB 테이블 구조
- toys: id, name, description, category, image_url, status(available/rented/maintenance)
- toy_rentals: id, toy_id, member_id, child_id, rented_at, due_date, returned_at, status(rented/returned/overdue), note
